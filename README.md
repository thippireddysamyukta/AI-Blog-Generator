# AI Blog Generator

An AI-powered blog writing tool using Google's Gemini API and Gradio. 
Generates blog posts based on topic, audience, tone, word count, and style.

## Features
- Customizable topic, audience, and tone
- Adjustable word count
- Multiple blog styles (Informative, Storytelling, SEO-Friendly)
- Simple Gradio web interface

## Tech Stack
- Python
- Google Gemini API
- Gradio

## Setup
```bash
pip install -r requirements.txt
```

## Usage
```bash
python blog_generator.py
```
Set your Gemini API key in the `client = genai.Client(api_key="YOUR_KEY")` line before running.
